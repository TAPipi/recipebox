# Recipe Box

A small React recipe organiser for screenshot OCR, pasted recipes, editing, and cook mode.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publish on GitHub Pages

This project is set up as a Vite app. The simplest route is:

1. Create a new GitHub repo
2. Upload these files
3. In the repo, enable GitHub Pages via GitHub Actions
4. Push to `main`
